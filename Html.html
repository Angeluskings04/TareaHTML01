Origen de HTML

La historia de HTML, a pesar de ser relativamente reciente, es vasta e increíblemente fascinante. A continuación, ofrecemos un breve resumen de este trayecto. El nacimiento de HTML data de 1980, cuando el físico Tim Berners-Lee, miembro del CERN (http://www.cern.ch/) (Organización Europea para la Investigación Nuclear), sugirió un innovador sistema de "hipertexto" para compartir documentos. Aunque la idea de "hipertexto" ya se había ideado unos años antes para facilitar el acceso a información complementaria en documentos electrónicos, estos sistemas terminaron siendo, de forma inesperada, los antecesores de los actuales enlaces web. Tim Berners-Lee, tras perfeccionar su sistema de "hipertexto", lo presentó en un evento destinado a innovar en sistemas de "hipertexto" para Internet, que era muy diferente al Internet actual. En colaboración con el ingeniero Robert Cailliau, Berners-Lee refinó su idea y juntos propusieron el WorldWideWeb o W3, que, como se puede observar, fue el proyecto seleccionado en el evento.

Principales versiones

  1. HTML (1991)
    • Esta fue la primera iteración de HTML creada por Tim Berners-Lee.
    • Contenía solo 18 elementos de marcado.
    • Estos elementos básicos permitían la estructuración de documentos y la creación de enlaces, y sentaron las bases para el desarrollo futuro del lenguaje.
  2. HTML 2.0(1995)
    • Primera versión estandarizada y oficial de HTML.
    • Introdujo nuevos elementos y atributos, muchos de los cuales todavía están en uso hoy en día.
    • Incluyó soporte para formularios, permitiendo la interacción del usuario con páginas web.
  3. HTML 3.0(1997)
    • Introdujo características como tablas, applets y texto centrado.
    • Sin embargo, debido a su ambición y la rapidez con la que se propuso, fue superado por la siguiente versión antes de ser ampliamente adoptado.
  4. HTML 3.2
    • Refinó y formalizó muchos de los elementos introducidos en 3.0.
    • Introdujo soporte para scripts (como JavaScript), lo que permitió una mayor interactividad en las páginas web.
    • Añadió más soporte para la estilización visual, aunque muchos de estos atributos de estilo luego fueron desaprobados en favor de usar CSS.
  5. HTML 4.0 (1997) y HTML 4.01 (1999)
    • Introdujo una distinción entre HTML "estricto" (sin presentaciones estilizadas) y HTML "transicional" (con estilizaciones). Mejoró la accesibilidad y la internacionalización.
    • Fomentó la separación entre estructura y presentación, animando a los diseñadores a usar hojas de estilo en cascada (CSS).
    • Introdujo el doctype, lo que permite a los navegadores interpretar y renderizar correctamente el contenido.

Estas primeras versiones sentaron las bases para el desarrollo web moderno. Cada versión trajo consigo una expansión de las capacidades de HTML, permitiendo a los diseñadores y desarrolladores crear experiencias web más ricas y dinámicas.

Etiquetas comunes y otras mas utiles

  • `<h1>`, `<h2>`,... `<h6>`:Encabezados de nivel 1 al 6.
  • `<br>`: Salto de línea.
  • `<a>`:Enlace o ancla.
  • `<head>`:Contiene metadatos y enlaces a hojas de estilo y scripts.
  • `< tfoott >` : Pie de la tabla.
  • `< ins>`: Texto insertado.
